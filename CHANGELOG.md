# Changelog

All notable changes to **Unpack & Play 3D** will be documented here.

## [0.2.0] - 2025-10-27
- Added diagnostics panel (FPS, draw calls, GPU) with `D` shortcut and wheel zoom capping.
- Implemented grid snap toggle (`G`), fine rotation (Alt+wheel), delete & undo/redo history.
- Integrated profiles with autosave, JSON export/import, and themed room recoloring.
- Added screenshot download, ambient particle field (respects reduced motion), touch gesture hints.
- Refreshed splash/menu HUD with big UI mode, settings, credits, and accessibility tweaks.
- Introduced the design palette for painting walls, floors, trim, rugs, and string lights with per-profile persistence.

## [0.1.0] - 2025-10-27
- Project scaffolding established (tooling, documentation baselines).
- Initial Three.js prototype for Unpack & Play experience.
